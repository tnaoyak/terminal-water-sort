# Terminal Water Sort

Terminal Water Sort は、ターミナル上で遊べるウォーターソート系パズルゲームです。

## 主な機能
- 縦向きボトル表示（Unicode罫線）
- ANSIカラーによる色分け表示
- `1 2` 形式のシンプルな操作入力
- ボトル10本構成のランダムステージ生成
- `r` `n` `help` `q` コマンド対応

## 動作環境
- Node.js 20 以上
- npm

## インストール

### ローカル開発
```bash
npm install
```

### npm公開後のグローバルインストール
```bash
npm install -g terminal-water-sort
```

## 実行方法

### 開発モード
```bash
npm run dev
```

### ビルドして実行
```bash
npm run build
npm run start
```

## 遊び方
- 目的: すべてのボトルを「空」または「単色で満杯」にそろえる
- 移動入力: `1 2` は「1番ボトルから2番ボトルへ注ぐ」
- 注ぎ替え可能条件:
  - 移動元が空でない
  - 移動先に空きがある
  - 移動先が空、または移動先の最上層色が移動元最上層色と同じ
- 最上層に連続する同色は、可能な範囲でまとめて移動する

## コマンド
- `1 2` 1番から2番へ注ぐ
- `r` 現在ステージをリスタート
- `n` 新しいランダムステージを生成
- `help` コマンド一覧を表示
- `q` ゲームを終了
